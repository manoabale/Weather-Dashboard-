# Weather-Dashboard-
A simple weather lookup app using predefined sample data.

🚀 Features

• Search weather by city
• Display temperature and condition
• Clean card UI


🧩 Tech Stack

• HTML
• CSS
• JavaScript


📂 Project Structure

index.html
style.css
script.js


📸 Preview

Shows a search bar and a weather result card.

🧠 How It Works

User enters a city name, and the app checks a fake weather dataset and displays the result.
